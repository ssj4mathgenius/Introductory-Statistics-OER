The Normal Distribution

The normal distribution, a continuous probability distribution, is one of the most widely used and studied distributions. Its bell-shaped curve appears in various fields such as psychology, business, economics, and the sciences. Examples include IQ scores, real estate prices, and more. It is essential for inferential statistics and probability analysis.

The normal distribution is characterized by a symmetric, bell-shaped curve when plotted. It is defined by two parameters: the mean (\( \mu \)) and the standard deviation (\( \sigma \)). The total area under the curve is equal to 1 making it a density curve which sets up that same connection between area and probability we observed with uniform distributions.

Key Features

  • The graph is symmetric around the mean (\(\mu\)), which is also the median and mode.
  • It is defined by two parameters: the mean (\(\mu\)) and the standard deviation (\(\sigma\)).

  • The density curve for the normal probability distribution is graphed by using the following formula:

    \[ f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{1}{2} \left( \frac{x-\mu}{\sigma} \right)^2} \]

    Note: It is not necessary to memorize this formula.

  • Since it has a more complicated bell-shape, we will have to rely on technology to find values associated with it.

The Standard Normal Distribution

Definition: Standard Normal Distribution

The standard normal distribution is a special case of the normal distribution where the mean is 0 and the standard deviation is 1. Values in this distribution are represented as z-scores.

Finding the area under the normal distribution is not quite as easy as it was for uniform distributions so we will heavily be relying on technology to help us find the area under these more complicated bell-shapes. The following tool in GeoGebra can both be used to find areas/probabilities as well as horizontal values ($z$-scores for standard normal distributions and $x$-values for non-standard normal distributions).

This is an embedded calculator designed to help with understanding normal distribution concepts in statistics.

This button opens the normal distribution calculator in a new tab for better accessibility and usability.

In order to use the applet, it is best to first choose raw data or z-scores where we can enter our horizontal value(s) (denoted as $x$ or $z$ in the applet). Then we can choose one of the "Tail Types":

  • Left-Tail: Use when normal distributions are shaded to the left side
  • Right-Tail: Use when normal distributions are shaded to the right side
  • Two-Tail: Use when normal distributions are shaded outward on two sides (this will not be used in this section)
  • Between: Use when normal distributions are shaded between two horizontal $x$ values.

Then, depending on the problem, we will have to change the "Input Types:

  • Data: Use when finding the area/probability from a known $z$ or $x$ value
  • Area/Probability: Use when the area/probability/percentage is given and we need to find the $z$ or $x$ value the cuts off this proportion of the normal distribution

We will practice using this tool in the the example problems that follow.

Applications of Normal Distributions

For any normal distribution problems that have a mean and standard deviation that are different from $0$ and $1$, respectively, it will be very similar to calculate the respective probabilities/areas using the GeoGebra applet. The only parameters we will need to change are the $\mu$ (mean) and $\sigma$ (standard deviation) parameters to find the solution for any application type problems.
