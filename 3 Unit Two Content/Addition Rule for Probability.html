Addition Rule for Probability

The Addition Rule for Probability "OR"

The addition rule calculates the probability that either event \(A\), event \(B\), or both occur. This rule is formally expressed as: \[ P(A \text{ or } B) = P(A) + P(B) - P(A \text{ and } B). \] This may seem a little hard to understand so we will break this down with an example below. The primary idea to focus on first is the key word in this section is "or," indicating that either event \(A\), event \(B\), or both can occur in a single trial. Let's look at an example of how we would find a compound probability (a probability where two or more events are involved) involving the word "OR".

Example : Using the Addition Rule

Refer to the table below and find the probability of getting a positive test result OR a subject who lied when one of the 85 test subjects is chosen.

Table 1: Polygraph Results
Test Result No (Did Not Lie) Yes (Lied)
Positive Test Result 12 37
Negative Test Result 30 6

This table summarizes the results of a polygraph test, showing counts for positive and negative test results against actual lying and not lying behavior.

Solution

If we want to solve this, we will simply count the number of subjects who tested positive or lied making sure that we do not count subjects twice. If we are applying the formal formula for finding \( P(A \text{ or }B) \) it would look like the following: \[ P(\text{tested positive OR lied}) = P(\text{tested positive}) + P(\text{lied}) - P(\text{overlap of positive and lied})\] which would reduce to \[ P(\text{tested positive OR lied}) = \dfrac{12+37}{85} + \dfrac{37+6}{85} - \dfrac{37}{85}=\dfrac{12+37+37+6-37}{85}=\dfrac{55}{85}\] Sum the relevant counts and then divide by the total number of subjects. We get: \[ P(\text{tested positive or lied}) = \dfrac{12+37+6}{85}=\dfrac{55}{85} \approx 0.647. \]

So we can see that to apply the addition rule we will be adding the probability of event A and the probability of event B and, if there is any overlap that causes double-counting, compensate for it by subtracting the probability of outcomes that are included twice. This approach is reflected in the above formal addition rule. However, it may be easier to consider this more intuitive approach documented below when working with these problems in this introductory course.

Definition: Intuitive Approach to Addition Rule

To find \(P(A \text{ or } B)\), add the number of ways event \(A\) can occur and the number of ways event \(B\) can occur, ensuring every outcome is counted only once. Then, divide this sum by the total number of outcomes in the sample space.

Example : Practice Problems

Using the polygraph results table provided earlier, solve the following problems:

  1. Find \( P( \text{tested negative or lied}) \).
  2. Find \( P( \text{didn’t lie or positive test}) \).
  3. Find \( P( \text{positive test or negative test}) \).

Solution

  1. To find \(P(\text{tested negative or lied})\), sum the probabilities of each relevant event and subtract the overlap: \[ P(\text{negative or lied}) = \dfrac{36}{85} + \dfrac{43}{85} - \dfrac{6}{85} = \dfrac{73}{85} \approx 0.8588. \] You could alternatively add the number of negative results or results where the individuals lied, but make sure not to add the "6" people who both lied and had a negative result twice: \[ P(\text{negative or lied}) = \dfrac{30+6+37}{85} = \dfrac{73}{85} \approx 0.8588 \]

  2. To find \(P(\text{didn’t lie or positive test})\), sum the probabilities of each relevant event making sure not to double count the "12" individuals that both didn't lie and had a positive result: \[ P(\text{didn’t lie or positive test}) = \dfrac{37+12+30}{85} = \dfrac{79}{85} \approx 0.9294. \]

  3. To find \(P(\text{positive test or negative test})\) we will simply add every outcome since all the outcomes are classified as either positive or negative: \[ P(\text{positive test or negative test}) = \dfrac{30+6+37+12}{85} = 1. \] If we were using the formal rule for finding \( P(A \text{ or } B) \), something interesting would happen in this case since these two events do not share any overlap. We would have gotten \[ P(\text{positive result or negative result}) = \dfrac{12+37}{85} + \dfrac{30+6}{85} - \dfrac{0}{85} = \dfrac{49}{85}+\dfrac{36}{85}-0 = 1. \]

Disjoint Events and Complements

Whenever events \(A\) and \(B\) are disjoint, \(P(A \text{ and } B)\) becomes zero in the formal addition rule which happened for our previous problem.

Definition

Events \(A\) and \(B\) are disjoint (or mutually exclusive) if they cannot occur at the same time, meaning they do not overlap. For example, if we had these two events

  • Randomly selecting an inaccurate order from McDonald's
  • Randomly selecting an accurate order from McDonald's
we would say these two events are disjoint since it is impossible to select an order that is both accurate and inaccurate.

For disjoint events \(A\) and \(B\), we have a more reduced "addition rule" for finding probabilities of "OR" problems (as was observed in the previous example) like \[ P(A \text{ or } B) = P(A) + P(B). \] since there is no overlap to subtract. However, instead of blindly applying the formula, it is more effective to understand and intuitively apply the addition rule.

Key Points of the Addition Rule

  1. To find \(P(A \text{ or } B)\), associate the word "or" with addition.
  2. To calculate \(P(A \text{ or } B)\), add the number of ways \(A\) can occur and the number of ways \(B\) can occur, being careful to avoid double-counting.

The most common way we will encounter disjoint events is when looking at an event's complement, which is denoted often with a "bar" over the event itself.

Definition: Complement of an Event

For any event \( A \), it's complement \( \overline{A} \) is all outcomes in which event \( A \) does not occur. Common sense dictates the following principle: We are certain (with probability 1) that either an event \( A \) occurs or it does not occur. Thus, it follows that: \[ P(A \text{ or } \overline{A}) = 1 \]

Because events \( A \) and \( \overline{A} \) must be disjoint, we can use the addition rule to express this principle as: \[ P(A \text{ or } \overline{A}) = P(A) + P(\overline{A}) = 1 \] This result of the addition rule leads to the following three equivalent expressions, which are just different forms of the same principle:

Rule of Complementary Events

  • \( P(A) + P(\overline{A}) = 1 \)
  • \( P(\overline{A}) = 1 - P(A) \)
  • \( P(A) = 1 - P(\overline{A}) \)

Example : Complement Probability

Assume \(P(\text{rain today}) = 20\%\). Write out the complement and calculate its probability.

Solution

The complement of \(P(\text{rain today})\) is \(P(\text{no rain today})\), and if we want to find the \(P(\text{no rain today}) \) we could use the second formula above: \( P(\overline{A}) = 1 - P(A) \). Writing \(P(\text{rain today})\) as a decimal we know \(20\% = 0.20\) giving us \[ P(\text{no rain today}) = 1 - P(\text{rain today}) = 1 - 0.2 = 0.8 \text{ or } 80\%. \]
