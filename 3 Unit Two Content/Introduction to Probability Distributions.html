Introduction to Probability Distributions

Probability distributions describe what will theoretically happen instead of what actually happened. They serve as theoretical models and when working with them we will use a different symbol for the mean and the standard deviation. When working with sample information we used \( \bar{x}\) for the mean and \( s \) for the standard deviation, but for probability distributions we will use the population mean (represented by \( \mu \)) and standard deviation (represented by \(\sigma\)) symbols.

Definition: Random Variable

A random variable is a variable (typically represented by the variable \(X\)) that has a single numerical value, determined by chance, for each outcome of a procedure.

Below are some examples of a random variable:

  • The number of eggs that a hen lays in a day: each day is different so this could be 0, 1, etc (making it a discrete random variable)
  • The amount of milk a cow produces in one day: this could be 2 pts, 1.2 liters, etc each day (making it a continuous random variable)
  • The number of movies currently playing in U.S. theaters (another discrete random variable since movies are countable objects)

Definition: Discrete Random Variable

A discrete random variable has either a finite number of values or a countable number of values, where “countable” refers to the fact that there might be infinitely many values, but they can be associated with a counting process, so that the number of values is \(0, 1, 2, 3, \dots\).

Definition: Continuous Random Variable

A continuous random variable has infinitely many values, and those values can be associated with measurements on a continuous scale without gaps or interruptions.

Example : Classifying Random Variables

Tell if each of the following random variables is a discrete random variable or a continuous random variable:

  • The number of people now driving a car in the United States
  • The height of the last airplane that departed from JFK Airport in New York City
  • The total amount (in ounces) of soft drinks that a person consumed in the past year

Solution

  • Discrete since the number of people is countable (0, 1, 2, etc)
  • Continuous since distances are always measured from a continuous scale
  • Continuous since volumes are always measured from a continuous scale

From this point forward we will mostly be focusing on discrete random variables in this section as we cover probability distributions and, later, binomial distributions, but in future sections we will cover distributions that are continuous (such as the Uniform and Normal distributions).

Probability Distributions

Definition: Probability Distribution

A probability distribution is a distribution that gives the probability for each value of the random variable. It is often expressed in the format of a graph, table, or formula. Many times \(x\) will be used for the various values of the random variable and \(P(x)\) will represent the probabilities for obtaining \(x\) number of outcomes.

A visual representation can often enhance our understanding of what a probability distribution entails. In essence, a probability distribution (in a table format) shares similarities with a frequency distribution mentioned in previous sections, but with a key difference: instead of counting occurrences, it provides the likelihood of various outcomes. Probability distributions can be very helpful since they list out all possible outcomes meaning we will not have to calculate individual probabilities like the previous sections. For instance, if we wanted to know the "probability of getting exactly two green peas in an experiment with 5" then we know from the table this probability would be 0.088 or 8.8%. Similarly, if we wanted to know the "probability of getting 2 or fewer peas with green" then we would know to add 0.088, 0.015, and 0.001 to get the sum of 0.104 or 10.4% chance to get 2 or fewer green peas.

Verifying if a Table is a Probability Distribution

When determining whether a table represents a probability distribution, there are two main conditions to check in order for a distribution to actually be a probability distribution:

  1. All probability \(P(x)\) values must be between \(0\) and \(1\), inclusive.
  2. The sum of all probabilities \(P(x)\) must equal \(1\).

Significant Values

If you know the mean and standard deviation, you can identify “significant” numbers of successes using these boundaries:

  • Significantly low values are \(\mu - 2\sigma\) or lower.
  • Significantly high values are \(\mu + 2\sigma\) or higher.

This table represents the number of girls (\(x\)) born in a family of 8 children, along with the probabilities (\(P(x)\)).

  • Find the probability of getting exactly 6 girls in 8 births \(P(x=6)\).
  • Find the probability of getting 6 or more girls in a family of 8 children \(P(x \geq 6)\).
  • Find the mean and the standard deviation for the probability distribution above.

Solution

  1. \(P(x=6) = 0.1094\) which is directly from reading the table.
  2. Adding the values from the table for \(x=6,7,\) and \(8\) we would get \(P(x \geq 6) = P(x=6) + P(x=7) + P(x=8) = 0.1094 + 0.0313 + 0.0039 = 0.1446\)

  3. The mean and the standard deviation can be calculated using technology (eg., GeoGebra)

    Results: \(\mu = 4.0\), \(\sigma = 1.4\).

  4. Significant values are outside the range \(\mu - 2\sigma\) to \(\mu + 2\sigma\):

    \[ \mu - 2\sigma = 4.0 - 2(1.4) = 1.2, \quad \mu + 2\sigma = 4.0 + 2(1.4) = 6.8 \]

    Since \(6\) is within this range, 6 girls is not considered a significant number of girls.

Alternative Way to Identify Significant Values

You can also use probability values to identify significant values by adding probabilities and comparing them to 5%. A value \(x\) is considered:

  • Significantly low if \(P(x \text{ or fewer}) < 0.05\).
  • Significantly high if \(P(x \text{ or more}) < 0.05\).

Example

Complete the statements below using the given information:

  1. Say that we add the probabilities of \(2\) or fewer (so \(P(0) + P(1) + P(2)\)) and get the value \(0.0912\). Then we know that “\(2\) is not a significantly low number of successes since the probability of \(2\) or fewer is not less than \(5\%\).”
  2. Say that we add the probabilities of \(7\) or more for a random variable with \(9\) trials (so \(P(7) + P(8) + P(9)\)) and get the value \(0.0124\). Then we know that “\(7\) is a significantly high number of successes since the probability of \(7\) or more is less than \(5\%\).”

Example

Use the table in the previous example to determine if \(6\) girls is a significantly high number of successes by using probabilities.

Solution

Add the probabilities \(P(6) + P(7) + P(8)\):

\[ P(6) + P(7) + P(8) = 0.1094 + 0.0313 + 0.0039 = 0.1446 \]

Since \(0.1446 > 0.05\), \(6\) girls is not a significantly high number of successes.
