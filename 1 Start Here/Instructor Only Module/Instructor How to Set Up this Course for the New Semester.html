Instructor: How to Set Up this Course for the New Semester

This course was created by a course developer. The content and assignments are set up and ready to go. It is important that you add your information to the course and update any dates that are listed. Some key updates are:

  • Add your name to the Syllabus. Read it over and add or update any course policies like late work or participation expectations.
  • Add your name to the welcome announcement.
  • Add your name, contact info, and a short bio to the Instructor Information page.
  • Review the course schedule. Update/adjust to meet your needs for the semester. Be sure the dates for all assignments match the course schedule.
