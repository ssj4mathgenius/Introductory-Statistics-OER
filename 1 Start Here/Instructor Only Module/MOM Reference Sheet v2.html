1) Common Control — Authoring Snippets

Frequently used code when scripting problem logic in the Common Control area.

Conditional assignment

$x = 4 if ($a == 2);
Assigns 4 to $x only when $a == 2. For multi-line logic, use a full if {...} else {...} block.

Multipart answer types

$anstypes = array("number", "choices");
Types apply per part. Common: number, choices, multans, calculated, string, essay.

Part weights

$answeights = array(0.5, 0.5);
Controls scoring per part. Example: array(0.4,0.4,0.2).

Declare answers

$answer[0] = "5";  // or single-part: $answer = "5";

Choices (per part)

$choices[1] = array("Yes","No");
Ensure the corresponding $anstypes[x] is "choices".

Display formatting

$displayformat = "decimal"; // "mixednumber", "fraction", "scinot", "sigfig", "logic", "list", "string", "typein", "noinput"

Precision & tolerances

$reqdecimals = 2;
$abstolerance = 0.01;
$reltolerance = 0.001;
Use indexed forms (e.g., $reqdecimals[1] = 3;) to target a specific part in multipart questions.

Load macro libraries

loadlibrary("stats");
// or multiple: loadlibrary("stats,fractions");
Examples: mean(), stdev(), percentile(), linreg() from stats.

Randomization helpers

$x = rand(1, 10);          // random int 1..10
$r = rand(100, 200)/100;  // random float 1.00..2.00
$alpha = $alphas[rand(0,2)]; // pick from array

Distinct randoms

$a,$b = diffrands(1,10,2);

Rounding

$z = round($r, 2);

2) Question Text — Presentation Snippets

HTML and inline math you’ll place in the Question Text editor.

Answer box placement

<p>What is 2 + 2? $answerbox</p>

Math expression (ASCIIMath)

`x^2 + 4x + 4`

Hint location (commented)

<!-- <p>$hintloc</p> -->

Apply shared CSS from another problem

// Common Control
includecodefrom(1610504);
// Top of Question Text
$css
Place $css as the first line of Question Text (no wrapping tags). Then use provided classes like .problem-mom, .note-mom, .alist-mom, .redtext-mom.
